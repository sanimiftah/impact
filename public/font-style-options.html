üí° Implementation Tips

  • Logo: Use font-weight 700 (bold) for brand name
  • Headlines: Use font-weight 800-900 for main headings
  • Navigation: Use font-weight 600-700 for menu items
  • Mobile: Slightly smaller sizes but same weights
  • Loading: Include font-display: swap for better performance

üìù CSS Implementation

/* Add to your HTML head */
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700;800;900&display=swap" rel="stylesheet">

/* CSS Classes */
.logo-text {
  font-family: 'Inter', sans-serif;
  font-weight: 700;
  font-size: 1.3rem;
  color: #374151;
}

.hero-title {
  font-family: 'Inter', sans-serif;
  font-weight: 800;
  font-size: 2.5rem;
  line-height: 1.1;
}

/* Mobile Responsive */
@media (max-width: 768px) {
  .logo-text { font-size: 1.1rem; }
  .hero-title { font-size: 2rem; }
}
